# Node Restful API - MySQL

## 1. Install MySQL
	- Download & install XAMPP or else.
	- Execute the query (create the db).
	
## 2. Open terminal (third):
	- npm install
	- gulp
	
## 3. Run - http://localhost:3000/api/
	- GET: /users
	- POST: /users
	- GET: /users/id
	- PUT: /users
	- DELETE: /users/email
	
	Format:
	{
	"email": "example@asd.com",
	"password": "pass"
	}
